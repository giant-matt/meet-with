## 1. 주최자 (약속 만들기)

### 1-1. 약속 생성 페이지로 이동

메인 화면에서 **새 약속 만들기** 버튼을 클릭합니다.

### 1-2. 기본 정보 입력

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 약속 제목 | O | 약속의 이름 (예: 교육품질 제고를 위한 위원회) |
| 내 이름 | O | 주최자 본인의 이름 |
| 이메일 | O | 내 약속 목록 조회 시 사용됩니다 |
| 설명 | X | 약속에 대한 간단한 안내 |

### 1-3. 응답 방식 선택

두 가지 방식 중 하나를 선택합니다.

- **날짜 + 시간**: 참여자가 가능한 날짜와 시간대를 모두 선택합니다.
- **날짜만**: 참여자가 가능한 날짜만 선택합니다. (시간 세부 설정 없음)

### 1-4. 후보 날짜 선택

달력에서 후보 날짜를 클릭하여 선택합니다.

- 날짜를 한 번 더 클릭하면 선택이 해제됩니다.
- **Shift + 클릭**으로 날짜 범위를 한번에 선택할 수 있습니다.
- 과거 날짜는 선택할 수 없습니다.

### 1-5. 시간 범위 설정 (날짜 + 시간 방식인 경우)

참여자가 선택 가능한 시간 범위를 지정합니다.

- **시작 시간**: 기본값 09:00
- **종료 시간**: 기본값 18:00
- 30분 단위로 설정할 수 있습니다.

### 1-6. 개인정보 수집 동의

개인정보처리방침 동의 체크박스에 체크합니다. (필수)

### 1-7. 약속 만들기

**약속 만들기** 버튼을 클릭하면 약속이 생성됩니다.

- 생성 후 약속 상세 페이지로 자동 이동합니다.
- 주최자도 바로 본인의 가능 시간을 입력할 수 있습니다.

### 1-8. 링크 공유

약속 상세 페이지 하단의 **링크 복사** 버튼을 클릭하여 링크를 복사합니다.
복사한 링크를 참여자들에게 메신저, 이메일 등으로 공유하세요.

### 1-9. 결과 확인

참여자들이 시간을 입력하면 실시간으로 반영됩니다.

- **히트맵**: 색이 진할수록 많은 사람이 가능한 시간입니다.
- **최적 시간 추천**: 가장 많은 사람이 가능한 상위 3개 시간이 자동으로 표시됩니다.
- **참여자 목록**: 우측에서 개별 참여자를 클릭하면 해당 참여자의 응답만 강조 표시됩니다.

### 1-10. 약속 수정

약속 상세 페이지에서 **약속 수정** 링크를 클릭합니다.

- 약속 생성 시 입력한 이메일로 본인 확인이 필요합니다.
- 날짜를 삭제하면 해당 날짜의 참여자 응답도 함께 삭제되니 주의하세요.

### 1-11. 내 약속 관리

상단 메뉴의 **내 약속**을 클릭하고, 이메일을 입력하면 본인이 만들거나 참여한 약속 목록을 조회할 수 있습니다.

- 본인이 만든 약속은 삭제할 수 있습니다.

## 2. 참여자 (시간 입력하기)

### 2-1. 링크 접속

주최자에게 받은 링크를 클릭하여 약속 페이지에 접속합니다.

### 2-2. 가능시간 입력 시작

**내 가능시간 입력하기** 버튼을 클릭합니다.

### 2-3. 이름 입력

참여자 정보 팝업이 나타나면 이름을 입력합니다.

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 이름 | O | 다른 참여자에게 표시되는 이름 |
| 이메일 | X | 내 약속 목록 조회 시 사용됩니다 |

- 이미 응답한 사람의 이름이 표시되며, 본인 이름을 선택하면 기존 응답을 수정할 수 있습니다.
- **확인** 버튼을 누르면 시간 선택 화면으로 전환됩니다.

### 2-4. 가능한 시간 선택

표에서 본인이 가능한 시간을 선택합니다.

- **클릭**: 셀을 하나씩 선택/해제합니다.
- **드래그**: 여러 셀을 한번에 선택할 수 있습니다.
- 선택된 셀은 **파란색**으로 표시됩니다.
- 배경에 연하게 보이는 색은 다른 참여자들이 선택한 시간입니다.

### 2-5. 제출

시간 선택이 끝나면 **제출하기** 버튼을 클릭합니다.

- 제출 후 히트맵에 본인의 응답이 즉시 반영됩니다.
- 같은 브라우저에서 다시 접속하면 본인의 응답을 수정할 수 있습니다.

### 2-6. 결과 확인

모든 참여자의 응답이 히트맵으로 표시됩니다.

- 색이 진할수록 많은 사람이 가능한 시간입니다.
- 셀에 마우스를 올리면 해당 시간에 가능한 사람의 이름을 확인할 수 있습니다.
- **최적 시간 추천**에서 가장 많은 사람이 가능한 시간 상위 3개를 확인할 수 있습니다.
